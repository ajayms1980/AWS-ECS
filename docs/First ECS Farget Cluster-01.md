

# Let's Create First Fargate ECS

  <ul>
      <li>  <p> It should be found under Container Section of the Service ( or you can search service ECS in search textbox)</p></li>
   </ul>

## Step2- Getting Started Option in ECS 
   <ul>
       <li>  <p> Click on Getting Started Button </p></li>
   </ul>
   <img src="https://github.com/kmitsolution/AWS-ECS/blob/main/images/OpenECS-GettingStarted-1.PNG" width="700" height="300" />
   
## Step3- Create Container and Task 
   <ul>
        <li>  <p> Create Container (Define Container image)  and Task Definitation ( Define Task ) </p></li>
        <li> <p> Click on Edit Button to modify Container or Task Definitation </p></li>
        <li> <p> Click on Next Button  </p></li>
   </ul>
   <img src="https://github.com/kmitsolution/AWS-ECS/blob/main/images/Container and Task Definiation -3.PNG" width="600" height="400" />
 
 ## Step4- Define Service
